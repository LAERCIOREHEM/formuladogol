HOTFIX - Curadoria das fontes dos melhores momentos do Brasileirão

Objetivo
--------
Ajustar a busca POWER para NÃO publicar novos vídeos de canais aleatórios.

Regra aplicada:
1. O que já está vinculado no site fica preservado.
2. Quando o workflow GE TV encontrar o vídeo oficial/Globo do mesmo jogo, ele substitui o link antigo.
3. Daqui para frente, a busca POWER só publica automaticamente vídeos de:
   - GE TV / ge.globo / sportv / Premiere / Globoplay
   - CazéTV
   - Amazon Prime Video / Prime Video
4. Qualquer outro canal encontrado pelo POWER vai apenas para auditoria, sem aparecer no site automaticamente.

Arquivos alterados
------------------
- scripts/buscar_melhores_momentos_youtube_power.py
- scripts/gerar_relatorio_fontes_melhores_momentos.py
- dados-br/getv-config.json
- .github/workflows/buscar-melhores-momentos-youtube-power.yml
- .github/workflows/buscar-melhores-momentos-getv.yml

Como rodar agora
----------------
1. Suba os arquivos deste ZIP no GitHub, preservando as pastas.
2. Rode manualmente primeiro:

   Actions -> Buscar melhores momentos GE TV -> Run workflow
   modo: backfill
   rodada_inicio: 1
   rodada_fim: 18

   Essa execução é a melhor para tentar substituir links de outros canais por GE/Globo, porque ela consulta playlists da GE TV e é barata em quota.

3. Depois que terminar, consulte o arquivo gerado:

   dados-br/auditoria-fontes-melhores-momentos.json

   Ele mostrará:
   - quantos links efetivos são GE/Globo;
   - quantos são Amazon Prime Video;
   - quantos são CazéTV;
   - quantos ainda são de OUTROS sites;
   - quais jogos ainda estão usando OUTROS sites.

4. Só rode o POWER depois se houver rodada/jogo pendente e com limite baixo:

   Actions -> Buscar melhores momentos YouTube POWER -> Run workflow
   rodada_inicio: rodada desejada
   rodada_fim: mesma rodada
   max_buscas_total: 8 a 12
   max_consultas_por_jogo: 1
   min_confianca: 0.82
   ignorar_cache: false

Observação importante
---------------------
Este hotfix NÃO remove links antigos de canais não preferenciais.
A ideia é exatamente manter esses links enquanto não houver GE/Globo, CazéTV ou Amazon Prime Video para substituí-los.

A partir deste hotfix, novos links automáticos de canais fora da whitelist não entram mais no site.
