1. Cadastrar participantes

Digite um nome por linha. Ao gerar, cada pessoa nova recebe um PIN de 6 dígitos automático. Quem já está na lista mantém o PIN — pode adicionar nomes novos sem perder os antigos.

2. Enviar os acessos

NomePINStatus

O botão WhatsApp copia uma mensagem pronta com o nome, o PIN e o link — é só colar na conversa da pessoa. Resetar PIN gera um novo número (use se alguém perder). A coluna Status mostra quem já enviou o palpite.

Modo teste: a lista fica salva só neste navegador.

⚠️ Em produção (com Supabase), os PINs ficam guardados com criptografia no banco, nunca em texto puro. Esta tela é o atalho para o modo de testes no seu PC — no servidor, o cadastro roda pelo SQL/RPC descrito no PASSO-A-PASSO.
